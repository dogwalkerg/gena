## gena

导航网站生成器

![Test](https://github.com/x1ah/gena/workflows/Test/badge.svg) ![Lint](https://github.com/x1ah/gena/workflows/Lint/badge.svg)

---

### 安装

```asciidoc
go get github.com/x1ah/gena/cmd/gena
```

### 使用

```asciidoc
gena -c config.yaml -t webstack > index.html
```
